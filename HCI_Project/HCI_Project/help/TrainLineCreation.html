Line Edit/Create Page

This window is for line creation and line edit. When you create new line fields are empty and when you edit fields are filled.

This is window for line creation.

1 - Map where you can place pins for stops!

2 - Pin for placing on the map

3 - Here you enter line default price

4 - Here your enter addition percentage for first class

5 - Here all of the selected stations are listed

6 - Button for adding line

Process for creating new station is simple!

1 - First you place pins on the map to specify stops. You can only place pin on the existing stations!

2 - If you want to delete specific station, you can double-click on the pin on the map or on the list item in Stations list!

3 - Then you enter line price

4 - Then you enter addition percentage for first class

5 - When all steps are done you press Create Line button

This is how the window should look like when all fields are filled!

As told you can delete stations by Double-clicking pin or station in list. Here we deleted Belgrade stop.

Now we added new stop Valjevo.

If all field are entered correctly, this popup will show!

This is how window looks when you want to edit line. All fields are filled. Proccess is the same as creation process!

You can enter new price and percentage.

If you want to change stops, first you have to remove them.

Last step, press Save button

If all field are entered correctly, this popup will show!
