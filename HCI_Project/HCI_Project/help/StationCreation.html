Station Edit/Create Page

This window is for station creation and station edit. When you create new station fields are empty and when you edit fields are filled.

This is window for station creation.

1 - Map where you can place pin for station!

2 - Pin for placing on the map

3 - Here you enter stations Name

4 - Button for adding station

Process for creating new station is simple!

1 - First you move pin somewhere on the map

2 - If you want to change location, you have to delete already placed pin and you do that with double-click on it. Then you add new pin!

2 - Then you enter name

3 - When you add pin and enter name you press this button to add station!

This is how the window should look like when all fields are filled!

If all field are entered correctly, this popup will show!

This is how window looks when you want to edit station. All fields are filled. Proccess is the same as creation process!

You can enter new name if you want.

If you want to change location, you have to remove already placed pin and place new!
