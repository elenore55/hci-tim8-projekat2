Lines CRUD page

On this page you can add, preview, edit and delete lines!

1 - This is map where route will be displayed!

2 - This is button for opening line creation window!

3 - This is list of all lines. Here you can see line ID, Start Station, End Stations, Default Price, First Class Percentage and you can click on the station to focuse it on the map.

3.1 - If you want to edit line you can click on this Edit button! This Line Edit Window will open, with line information loaded!

3.2 - If you want to delete line you can click on this Delete button!

Preview of line

When you click on the line, map will display route of the line!

Line Creation

When you click on Add Line button, Line Creation Window will open!

When line is added it will show up in the list and will be displayed on the map!

Line Edit

When you click on the Edit button, Line Edit Window will open! When you save changes, list and map updates!

Line Deletion

When you click on the Delete button, window like this will open!

You can confirm or decline deletion. If you choose Yes, there are two possible outcomes. If station is free, it will be deleted! But if any of future departures of line is reserved or ticket is bought, new window will show.

When line is deleted this will popup!
