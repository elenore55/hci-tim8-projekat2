Stations CRUD page

On this page you can add, preview, edit and delete stations!

1 - This is map where you can see all stations marked!

2 - This is button for opening station creation window!

3 - This is list of all stations. Here you can see station ID and Name and you can click on the station to focuse it on the map.

3.1 - If you want to edit station you can click on this Edit button! This Station Edit Window will open, with station information loaded!

3.2 - If you want to delete station you can click on this Delete button!

Preview of station

When you click on the station (1), map will focuse on the station and marker(2) will become bigger than the others!

Station Creation

When you click on Add Station button, Station Creation Window will open!

When station is added it will show up in the list and will be focused on the map!

Station Edit

When you click on the Edit button, Station Edit Window will open! When you save changes, list updates!

Station Deletion

When you click on the Delete button, window like this will open!

You can confirm or decline deletion. If you choose Yes, there are two possible outcomes. If station is free, it will be deleted! But if any line conatins this station, you can not delete it!
