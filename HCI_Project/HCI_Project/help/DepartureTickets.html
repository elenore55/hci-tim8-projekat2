Departure tickets

On this page you can see the tickets that were sold for previously selected departure.

Part 1:

In this label you can see date and time of selected departure

Part 2:

In this table you can see all tickets for selected departure. The list can be sorted by multiple criteria by clicking on the criteria name in the table header.

Part 3:

If you want to see more details about the ticket, you can click on the ticket row, and you will see ticket details.
